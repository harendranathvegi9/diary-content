## github webhooks

max 20 webhooks for each event per repo

`curl -u "username" https://api.github.com`

`POST https://api.github.com/repos/:owner/:repo/hooks`

```json
{
  "name": "web",
  "active": true,
  "events": [
    "push"
  ],
  "config": {
    "url": "http://webhook.yourdomain.com/",
    "content_type": "json"
  }
}

```
