## upgrading ubuntu

10.04 LTS → 12.04 LTS → 14.04 LTS → 16.04 LTS
lucid → precise → trusty → xenial

check currently installed version
`lsb_release -a`

stop non-critical services
`service apache2 stop`
`service mongod stop`

```
sudo apt-get update
sudo apt-get dist-upgrade
sudo do-release-upgrade
```
